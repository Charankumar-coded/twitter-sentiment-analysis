# Twitter Sentiment Analysis

This project contains a simple Twitter sentiment analysis pipeline using a text classifier.

## Folder Structure

- dataset/ - sample Twitter sentiment data
- models/ - trained model files
- main.py - train the model
- predict.py - predict sentiment for a given tweet
- requirements.txt - dependencies

## Setup

1. Install dependencies:
   pip install -r requirements.txt
2. Train the model:
   python main.py
3. Predict sentiment:
   python predict.py
   python predict.py "so happy to see my friends today"
